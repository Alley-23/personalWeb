Git SSH配置

本文说明：

命令全部为截图标注，顺序阅读

准备工作：

安装git

注册GitHub账号

配置你的名称和邮箱

GitHub协议分为：https 和 SSH

https：使用https协议需要每次提交时输入用户名和密码（如果个人不介意可以使用）

SSH：使用SSH协议提交时可以不用输入用户名和密，但是需要配置

以下为SSH配置方法：

1.查看自己电脑是否有.ssh目录

.ssh目录下面有id_rsa和id_rsa.pub两个文件。

2.如果存在打开id_rsa.pub文件，复制里面的一串公匙

3.如果不存在需要生成SSH key

4.打开.ssh下的id_rsa.pub文件，复制这一串公匙

5.接着去GitHub设置SSH

6.在SSH and GPG keys 下新建一个SSH key

7.复制刚才生成的公匙到里面，然后添加

8.恭喜你到这一步已经添加成功了，现在就可以使用SSH协议了。
